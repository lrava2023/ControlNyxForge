# Nyx Stock Control

Nyx Stock Control es un sistema de control de inventarios desarrollado para optimizar la gestión de stock en comercios y emprendimientos. Permite administrar productos, controlar movimientos de entrada y salida, visualizar niveles de stock y reducir errores operativos. El proyecto está pensado con una arquitectura escalable, adaptable a distintos tipos de negocio y orientado a buenas prácticas de desarrollo de software.
